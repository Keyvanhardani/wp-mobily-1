== Installation ==

1. Upload the plugin Wp-Mobily files to the `/wp-content/plugins/` directory
1. Activate the Wp-Mobily plugin through the 'Plugins' menu in WordPress.
1. Select Themes For Devices under Settings > Wp-Mobily
